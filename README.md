LifeLine BD | Premium Blood Donation Platform

**LifeLine BD** is a modern, responsive, and user-centric web platform designed to bridge the gap between blood donors and those in urgent need. Built with a focus on speed and accessibility, this platform simplifies the process of finding life-saving blood donors across Bangladesh.

### 🚀 Key Features
* **Smart Donor Search:** Quickly filter and find donors based on specific blood groups and locations.
* **Instant Connectivity:** One-tap calling and direct WhatsApp messaging integration for immediate communication.
* **Real-time Database:** Powered by **Firebase**, ensuring that donor information and availability are updated instantly.
* **User Authentication:** Secure registration and login system for donors to manage their profiles and availability.
* **Interactive UI:** A premium, mobile-first design featuring a "Favorite" system to save frequent donor contacts.
* **Live Info Bar:** Displays real-time date, time, and weather updates for a personalized user experience.

### 🛠️ Tech Stack
* **Frontend:** HTML5, Tailwind CSS (Utility-first styling).
* **Backend/Database:** Google Firebase Realtime Database.
* **Icons & Fonts:** Font Awesome, Google Fonts (Noto Sans Bengali).
* **Hosting:** GitHub Pages.

### 💡 Mission
*"One bag of blood, one new life."* LifeLine BD aims to digitize the blood donation process, making it more efficient to save lives during emergencies.
